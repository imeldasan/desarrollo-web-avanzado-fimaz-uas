# Práctica 1 - Creación de Clases y Encapsulamiento en PHP

## Objetivo
Aplicar los fundamentos de la Programación Orientada a Objetos en PHP,
implementando una clase con atributos privados, constructor y métodos
de acceso siguiendo buenas prácticas de encapsulamiento.

## Descripción de la clase
Se creó la clase Usuario con:
- Atributos privados: nombre y correo.
- Constructor para inicializar los atributos.
- Métodos getters para obtener los valores.
- Métodos setters para modificar los valores.

## Instrucciones de ejecución
1. Abrir una terminal dentro de la carpeta `practica-1`.
2. Ejecutar el siguiente comando para iniciar el servidor integrado de PHP:
   php -S localhost:8000
3. Abrir el navegador y acceder a: http://localhost:8000/index.php
